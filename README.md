# SpaceFinder (SFA) 🚀

A premium, high-performance real estate marketplace built with **Expo**, **React Native**, and **Supabase**.

![Banner](https://img.shields.io/badge/Platform-iOS%20%7C%20Android%20%7C%20Web-blue?style=for-the-badge)
![Tech Stack](https://img.shields.io/badge/Stack-Expo%20%7C%20Supabase%20%7C%20TypeScript-black?style=for-the-badge)

## ✨ Features

- **Luxury Property Discovery**: High-performance lists and masonry grids.
- **Real-time Messaging**: Instant chat between property owners and seekers.
- **Push Notifications**: Stay updated on new messages and property status.
- **Role-based Access**: Custom flows for Property Owners and Seekers.
- **Secure by Design**: Row Level Security (RLS) managed at the database level.

## 🛠 Tech Stack

- **Frontend**: Expo (SDK 55), React Native, Expo Router.
- **Backend**: Supabase (PostgreSQL, Auth, Realtime, Storage).
- **State**: React Query & Zustand.
- **Styling**: Native CSS variables & Reanimated 4.

## 🚀 Quick Start

1. **Install dependencies**
   ```bash
   npm install
   ```

2. **Configure Environment**
   Create a `.env` file:
   ```env
   EXPO_PUBLIC_SUPABASE_URL=your_url
   EXPO_PUBLIC_SUPABASE_ANON_KEY=your_key
   ```

3. **Launch the app**
   ```bash
   npx expo start
   ```
